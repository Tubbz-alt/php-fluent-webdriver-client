BrowserDriverBuilder

class BrowserDriverBuilder

Provides a fluent interface for initializing Facebook RemoteWebDriver objects.

Usage:

$driver = (new BrowserDriverBuilder('http://localhost:4444/wd/hub'))
    ->withType('phantomjs')
    ->build()
    ->getRemoteWebDriver();
