ElementFindWithAssertions

class ElementFindWithAssertions

Allows synchronous assertions on elements.

Usage:

$browser->get($url)->getElement()->withId('the-id')
    ->assertThat()
    ->isEnabled()
    ->thenFind()
    ->asHtmlElement();
