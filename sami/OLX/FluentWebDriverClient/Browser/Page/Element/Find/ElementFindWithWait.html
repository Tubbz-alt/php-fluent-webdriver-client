ElementFindWithWait

class ElementFindWithWait

Allows asynchronous assertions on elements.

Usage:

$browser->get($url)->getElement()->withId('the-id');
    ->wait(3)
    ->toBePresent()
    ->thenFind()
    ->asHtmlElement();
